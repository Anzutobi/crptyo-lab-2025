## 实验内容

包括：

实验1（Classic）：古典密码（1-1维吉尼亚密码，1-2仿射希尔密码）

实验2（GF(2^131)）：有限域的实现

实验3（AES）：AES-128的实现（CBC模式）

实验4（SHA-256）：SHA-256的实现

实验0-3（bignum）：大数运算，是RSA、DLP及DSA的基础

实验5（RSA）：RSA加解密

实验6（DLP）：DLP的计算

实验7（DSA）：DSA签名和验证

实验8（PRN）：期末当堂实验，Pseudo-Random Number，伪随机数生成器（二重离散对数伪随机数生成器）。
